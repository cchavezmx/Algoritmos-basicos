Ejemplo Input - [3,6,1,8,2,3,6,3,2,5,6]

<!-- sopoiler regex -->
Ejemplo input -> [8,‘e’,7,2,‘a’,‘d’,‘f’,2,3,1,4,3]
Salida → [‘e’,‘a’,‘d’,‘f’]

Ejemplo input [1,4,7,2,4,1,9,4,0,2,4,5,12]
<!-- hacerlo sin Math.max  -->
